# Agent Skills

Este directorio está reservado para **vendorizar skills** que use el agente.

- Cada skill debe vivir en su propia carpeta.
- Cada carpeta de skill debe incluir un archivo `SKILL.md` con sus instrucciones.

## Estructura esperada

```text
.agents/skills/<skill-name>/SKILL.md
```

> Nota: en este directorio solo se añade estructura y documentación; no se debe añadir código de producto.
